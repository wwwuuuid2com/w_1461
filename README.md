<h3>源码简介：</h3>
<p>一款非常适合管理微信的活码，可创建微信活码，活码管理，每日扫描统计及域名管理，将设置好的活码图片下载下来，用于对外统一宣传<p>
<p>本系统采用是thinkphp3.2框架，完全开源，免授权，只需要安装的时候随便填写即可激活 比如填写：qiuma随便填<p>
<p>阀值100次啥意思？ 是这个码扫到100后会根据活码类型来显示。比如循环码就是扫到100归零循环<p>
